# Agentic AI Travel Planner

## Project Description
Agentic AI Travel Planner is an intelligent system that autonomously plans travel itineraries based on user preferences such as destination, budget, travel dates, and interests. The system uses agent-based decision-making to generate personalized travel plans.

## Features
- Autonomous itinerary generation
- Budget-aware travel planning
- Personalized recommendations
- Dynamic decision making using AI agents

## Technologies Used
- Python
- Large Language Models (LLMs)
- Agentic AI concepts
- APIs (optional)
- Streamlit / Jupyter Notebook

## Use Cases
- Personal travel planning
- Budget travel optimization
- Smart itinerary generation
